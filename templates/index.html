欢迎使用 BiliNote 批量收藏转写

  1. 从 Bilishelf 浏览器插件导出收藏文件（JSON 或 CSV）
  2. 点击上方 "选择收藏文件" 加载数据
  3. 选择 供应商、模型 和 转写配置
  4. 点击 "处理全部待处理" 或选中视频后 "处理选中"
📤 文件导出

🔄 同步任务到 BiliNotes

将批量工具处理的任务同步到 BiliNotes 前端，使其在 BiliNotes 历史列表中可见。

  1. 点击下方 "复制同步脚本"
  2. 打开 BiliNotes 页面（点击右上角 🔗 BiliNote 按钮）
  3. 按 F12 打开浏览器开发者工具，切换到 Console（控制台）
  4. 粘贴脚本并按 Enter 执行
  5. 刷新 BiliNotes 页面，任务将出现在历史列表中
