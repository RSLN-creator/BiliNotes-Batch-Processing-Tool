    在现有 app.py 中按以下顺序修改:

    1. 文件顶部 import 区追加 import re, import time（现有的 import json, pathlib 等保持不变）

    2. 在 BILINOTE_BASE_URL 和 GUI_PORT 常量之后新增:
       - 常量 OUTPUT_DIR = Path(os.getenv("OUTPUT_DIR", "./output"))
       - 常量 _CHECKPOINT_FILE = "_已处理.json"

    3. 在 OUTPUT_DIR 之后、Flask 应用 app = Flask(__name__) 之前新增内存状态变量:
       - _task_map: dict = {}  —— task_id 到 {bvid, title, folder} 的映射
       - _completed_tasks: set = set()  —— 已保存文件的 task_id 集合，防重复写入
       - _processed_bvids: set = set()  —— 已成功处理的 BV 号集合，用于 checkpoint 跳过

    4. 在文件末尾、if __name__ == "__main__" 块之前追加以下 4 个函数:

       sanitize_filename(name: str, max_len: int = 80) -> str
         - 对输入 name 执行 re.sub(r'[\\/:*?"<>|]', '_', name) 替换非法字符
         - 去除首尾空白字符和点号 (strip '. ')
         - 如果长度超过 max_len，截断并 rstrip('_ ')
         - 如果最终结果为空字符串，返回 "untitled"
         - 用途: DATA-05

       save_video_output(folder_name: str, bvid: str, title: str, result: dict, output_dir: Path) -> dict
         - 用 sanitize_filename 处理 folder_name（max_len=60）和 title（max_len=80）
         - 构造 safe_title = sanitize_filename(title)
         - 构造 safe_folder = sanitize_filename(folder_name, max_len=60)
         - 构造基础文件名: f"{bvid} - {safe_title}"
         - 创建目录: output_dir / safe_folder，使用 mkdir(parents=True, exist_ok=True)
         - 从 result 提取内容: md_content = result.get("markdown", "") or ""; transcript = result.get("transcript", {}) or {}; full_text = transcript.get("full_text", "") or ""
         - 写 3 个文件:
           (a) .md — md_content（笔记 Markdown）
           (b) _原文.txt — full_text（字幕全文）
           (c) _完整.json — json.dumps(result, ensure_ascii=False, indent=2)（API 完整返回）
         - 返回 dict {md: str路径, txt: str路径, json: str路径}
         - 用途: DATA-03 按收藏夹分目录, DATA-04 3 文件输出

       load_checkpoint(output_dir: Path = OUTPUT_DIR) -> dict
         - 构造 path = output_dir / _CHECKPOINT_FILE
         - 如文件不存在则返回空 dict
         - try: open + json.load → 返回 dict
         - except (json.JSONDecodeError, OSError): app.logger.warning + 返回空 dict
         - 用途: DATA-01 启动时加载已处理记录

       update_checkpoint(bvid: str, status: str, task_id: str, output_dir: Path = OUTPUT_DIR)
         - 构造 path = output_dir / _CHECKPOINT_FILE
         - path.parent.mkdir(parents=True, exist_ok=True) 确保目录存在
         - 调用 load_checkpoint(output_dir) 获取现有记录
         - records[bvid] = {"task_id": task_id, "status": status, "time": time.strftime("%Y-%m-%dT%H:%M:%S")}
         - 原子写入: tmp = path.with_suffix(".tmp"); json.dump(...); os.replace(str(tmp), str(path))
         - 用途: DATA-01 实时记录完成状态

    5. 在 4 个函数之后、if __name__ 之前添加模块初始化逻辑:
       - _checkpoint_data = load_checkpoint()
       - 遍历 _checkpoint_data.items(), 对 status == "SUCCESS" 的记录将其 bvid 加入 _processed_bvids
       - app.logger.info(f"Loaded {len(_processed_bvids)} processed videos from checkpoint")

       - 从 request.get_json(silent=True) 提取数据
       - 无效数据则 return ({"error": "无效的请求数据"}, 400)
       - 提取字段: bvid, title, folder, url, model_name, provider_id, style, extras
       - 构造 BiliNote 参数字典:
         - video_url = data.get("url", 默认 f"https://www.bilibili.com/video/{bvid}")
         - platform = "bilibili"（固定值）
         - quality = "medium"（固定值）
         - model_name = data.get("model_name", "")
         - provider_id = data.get("provider_id", "")
         - style = data.get("style", "detailed")
         - extras = data.get("extras", "")
       - 用 requests.post(BILINOTE_BASE_URL + "/api/generate_note", json=params, timeout=10) 提交到 BiliNote
       - 成功: 提取 task_id, 存入 _task_map[task_id] = {bvid, title, folder}, 返回 {"task_id": task_id}
       - 异常: app.logger.error, 返回 ({"error": f"提交到 BiliNote 失败: {e}"}, 502)

    2. GET /api/task-status/{task_id}
       - requests.get(BILINOTE_BASE_URL + f"/api/task_status/{task_id}", timeout=5)
       - 失败时返回 ({"status": "UNKNOWN", "error": str(e)}, 502)
       - 成功时解析 data = resp.json()
       - status = data.get("status", "UNKNOWN"), result = data.get("result") or {}
       - 核心逻辑: if status == "SUCCESS" and task_id not in _completed_tasks:
         - _completed_tasks.add(task_id)
         - 从 _task_map 获取视频上下文（ctx = _task_map.get(task_id, {})）
         - 调用 save_video_output(folder, bvid, title, result, Path(...)) 保存文件
         - 调用 update_checkpoint(bvid, "SUCCESS", task_id) 更新 checkpoint
         - data["files"] = save_video_output 返回的路径 dict
         - 任何异常: app.logger.error + data["file_error"] = str(e)（不阻塞正常返回）
       - 返回 jsonify(data)

    3. GET /api/checkpoint
       - 返回 {"processed": sorted(list(_processed_bvids))}
       - 供前端在启动时获取已处理 BV 号列表用于跳过逻辑 (DATA-02)
  </action>
