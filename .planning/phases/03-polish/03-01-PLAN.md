    === 1. POST /api/process-video（约第 118-128 行） ===

    找到 `resp = requests.post(f"{BILINOTE_BASE_URL}/api/generate_note", json=params, timeout=10)` 之后的代码块，

    将:
    ```python
    resp.raise_for_status()
    task_id = resp.json().get("task_id", "")
    ```

    替换为:
    ```python
    raw = resp.json()
    if raw.get("code") != 0:
        return jsonify({"error": raw.get("msg", "BiliNote API 返回错误")}), 502
    task_id = (raw.get("data") or {}).get("task_id", "")
    ```

    保留 resp.raise_for_status() 用于 HTTP 级错误检测（resp.json() 前确保 resp.status_code == 200）。修改后效果:
    - HTTP 错误 → try/except 捕获，返回 502
    - HTTP 200 但 code != 0 → 返回 502 + BiliNote 错误消息
    - 正常 → 从 data.task_id 提取

    === 2. GET /api/task-status/{task_id}（约第 132-162 行） ===

    找到 `def task_status(task_id):` 函数体。

    将:
    ```python
    resp = requests.get(f"{BILINOTE_BASE_URL}/api/task_status/{task_id}", timeout=5)
    resp.raise_for_status()
    data = resp.json()
    ```

    替换为:
    ```python
    resp = requests.get(f"{BILINOTE_BASE_URL}/api/task_status/{task_id}", timeout=5)
    resp.raise_for_status()
    raw = resp.json()
    if raw.get("code") != 0:
        return jsonify({"status": "UNKNOWN", "error": raw.get("msg", "BiliNote API 返回错误")}), 502
    inner = raw.get("data") or {}
    ```

    将函数体内所有 `data.` 引用替换为 `inner.`（共 4 处: data.get("status"), data.get("result"), data["files"], data["file_error"]）:

    - `status = data.get("status", "UNKNOWN")` → `status = inner.get("status", "UNKNOWN")`
    - `result = data.get("result") or {}` → `result = inner.get("result") or {}`
    - `data["files"] = files` → `inner["files"] = files`
    - `data["file_error"] = str(e)` → `inner["file_error"] = str(e)`

    修改 return 语句: `return jsonify(data)` → `return jsonify(inner)`

    修改后效果:
    - BiliNote 返回 {code, msg, data: {status, result, ...}} → inner = data → 提取 status/result → 返回 inner
    - BiliNote 返回 code != 0 → 返回 502 + 错误消息
