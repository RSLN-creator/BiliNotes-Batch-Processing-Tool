  <action>
    在项目根目录创建 `app.py`，包含以下内容：

    1. **导入模块**：`Flask, render_template, jsonify, request` 来自 `flask`；`webbrowser`；`json, csv, io, os`；`requests`；`threading.Timer`；`pathlib.Path`

    2. **Flask 应用初始化**：`app = Flask(__name__)`

    3. **配置常量**（放在文件顶部，方便用户调整）：
       - `BILINOTE_BASE_URL = os.getenv('BILINOTE_URL', 'http://localhost:3015')`
       - `GUI_PORT = int(os.getenv('GUI_PORT', '8765'))`

    4. **路由 '/'**：`@app.route('/')` → `return render_template('index.html')`

    5. **启动函数**：在 `if __name__ == '__main__':` 块中：
       - `Timer(1, lambda: webbrowser.open(f'http://localhost:{GUI_PORT}')).start()` 延迟 1 秒打开浏览器
       - 打印启动信息：`[*] BiliNote Batch Tool 启动中...`、访问地址和 BiliNote 地址
       - `app.run(host='127.0.0.1', port=GUI_PORT, debug=False)`

    6. **注意事项**：
       - 使用 `threading.Timer` 而非 `time.sleep`（不阻塞启动线程）
       - 不从 `.env` 加载（当前阶段不需要额外依赖）
       - 不要添加任何 API 路由（这些在 Plan 02 和 Plan 03 中添加）
       - 不要添加端口冲突检测（属于 Phase 3 EDGE-06）
  </action>
