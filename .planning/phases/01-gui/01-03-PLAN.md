## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Flask proxy → BiliNote API | 后端代理请求到 BiliNote `localhost:3015` |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-05 | Information Disclosure | /api/providers proxy | accept | 仅在 localhost 代理 BiliNote API；不传输敏感用户数据 |
| T-01-06 | Denial of Service | /api/providers, /api/providers/{id}/models | mitigate | `timeout=5` 防止请求挂起；`requests.RequestException` 捕获防止代理崩溃 |
| T-01-07 | Denial of Service | /api/check-connection | mitigate | `timeout=3` 短超时确保快速失败 |
| T-01-SC | Tampering | pip package installs | mitigate | Plan 03 不安装任何新包。全部使用已预装的 Flask + requests。 |

## 错误处理决策（D-04, D-06）

| 场景 | 前端行为 | 后端行为 |
|------|---------|---------|
| BiliNote 不可达 | 显示黄色横幅 + 禁用供应商下拉 | 返回 `{connected: false}` |
| 供应商列表为空 | 显示黄色横幅 + 提示去配置 | 返回 `{connected: true, providers: []}` |
| 选中供应商后无模型 | 显示黄色横幅 + 保持模型下拉禁用 | 返回 `{connected: true, models: []}` |
| 网络超时 | 显示黄色横幅 | 5 秒超时后抛出 RequestException |
